# Code of Conduct

Hurrah.tv is a small project. The expectations are simple.

## What we expect

- **Be kind.** Disagreements are fine; personal attacks are not.
- **Be specific.** If you're pointing out a problem, point to the file, line, or behavior — not the person.
- **Assume good intent.** Default to the charitable interpretation of what someone wrote.
- **Respect time.** Read the issue body before commenting. Run the code before saying it's broken.

## What we don't accept

- Harassment of any kind — sexual, racial, personal, or based on identity
- Sustained disruption of discussions
- Publishing someone's private information without permission

## Scope

This applies to everything under this repository — issues, pull requests, commit messages, discussions, and any linked community channels.

## Reporting

If you experience or observe behavior that violates this, contact the repo owner directly:

- GitHub: [@mkerchenski](https://github.com/mkerchenski)
- Email: see GitHub profile

Reports will be handled privately. Retaliation against anyone who reports in good faith is itself a violation.

## Consequences

For contributors: warning → temporary access removal → permanent removal, depending on severity.

The repo owner's decision is final.

---

This is a hobby project, not a large open-source community — so this doc stays short on purpose. If the project grows, we'll consider adopting the [Contributor Covenant](https://www.contributor-covenant.org/) for its broader coverage.
